Project

HackerBodega

A digital superstore for hackers and tinkerers. Find cyber gadgets, tools, and gear for ethical hacking and tech.

Type E-Commerce Site
Stack HTML / CSS / JavaScript
Status Live

Overview

HackerBodega is a curated online storefront aimed at the ethical hacking and maker community. The goal was simple: one place to find the hardware and gear that you actually want as a security researcher or tinkerer, without wading through consumer junk.

What I Built

  • Custom front-end built from scratch — no frameworks, no templates
  • Product catalog with category filtering
  • Responsive layout that works across mobile and desktop
  • Clean, fast-loading pages with minimal dependencies

Tech Stack
