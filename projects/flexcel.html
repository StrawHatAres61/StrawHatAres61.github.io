Overview

Flexcel is a desktop application used by competitive high school debaters to "flow" rounds — tracking arguments in real time across LD, Policy, and PF formats. The original app didn't run properly on Linux, so I forked it and rebuilt the platform-specific pieces to make it work natively.

The result is a stable, fully-featured Linux build that the debate community has adopted widely. With 8,000+ downloads and over 30,000 hours of logged usage, it's the most-used tool I've shipped.

What I Did

  • Ported the Electron app from macOS/Windows to Linux — resolved IPC, remote module, and renderer process compatibility issues
  • Replaced deprecated electron.remote with @electron/remote
  • Fixed Promise-based dialog APIs for file save/load
  • Guarded macOS-only lifecycle hooks behind platform checks
  • Configured the build pipeline to produce AppImage and .deb distributables
  • Maintained backwards compatibility with existing saved flow files
